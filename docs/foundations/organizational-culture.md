# 組織文化と心理的安全性

## 概要

DORAの研究は一貫して、技術的プラクティスだけでなく**組織文化がソフトウェアデリバリーパフォーマンスの強力な予測因子**であることを実証してきた。広木大地氏も「不確実性に向き合う組織」をエンジニアリング組織論の中核に据えている。

---

## 1. Westrum組織文化モデル

社会学者Ron Westrumによる組織文化の3類型モデル。

| 病的（Pathological） | 官僚的（Bureaucratic） | 生成的（Generative） |
|---|---|---|
| 権力志向 | ルール志向 | パフォーマンス志向 |
| 低い協力 | 中程度の協力 | 高い協力 |
| メッセンジャーを撃つ | メッセンジャーを無視する | メッセンジャーを訓練する |
| 責任を回避する | 狭い責任範囲 | リスクを共有する |
| 橋渡しを阻害する | 橋渡しを容認する | 橋渡しを奨励する |
| 失敗はスケープゴートに | 失敗は裁きに | 失敗は探求に |
| 新規性を潰す | 新規性は問題を生む | 新規性を実装する |

### DORAの実証データ

- 生成的文化を持つ組織は**30%高い組織パフォーマンス**を示す（2023年レポート）
- セキュリティプラクティスの採用を最も強く予測する因子は文化である（2022年レポート）
- 信頼性の高い低blame文化がセキュリティプラクティスの採用を促進（2022年）
- セキュリティプラクティスが堅固なチームはバーンアウトが**1.4倍少ない**（2022年）

## 2. 心理的安全性

Amy Edmondsonが提唱した概念。Google Project Aristotleでも最も重要なチーム特性として確認された。

広木大地氏は講演「心理的安全性とソフトウェア化する社会」で、心理的安全性がチームの効果的な機能の基盤であることを論じている。

### DORAとの関係

DORAのケイパビリティモデルにおいて：
- **Generative Organizational Culture** が文化的ケイパビリティの筆頭
- **Learning Culture** と組み合わせることで、失敗からの学習→改善サイクルが回る
- **Team Experimentation** には心理的安全性が前提条件

## 3. 変革型リーダーシップ

DORAの2017年レポートで「変革型リーダーシップが高パフォーマンス文化の醸成に重要な役割を果たす」ことが実証された。

Accelerateで特定された24のケイパビリティの一つ（#24 Transformational Leadership）として位置づけられている。
